================================================================
ALERJİ TAKİM — PROJE HARİTASI (map.txt)
Güncelleme: 2026-05-29
================================================================
Proje: Türkiye'ye özel Polen Takip + Alerji Asistanı uygulaması.
Stack: React (Vite) frontend · Node.js/Express backend · PostgreSQL
       · Groq AI (RAG + LLM chatbot) · Google Pollen API / Open-Meteo fallback

================================================================
BACKEND
================================================================

── GİRİŞ NOKTALARI ──────────────────────────────────────────────

backend/server.js

  • shutdown(signal)               SIGTERM/SIGINT ile düzgün kapatma

backend/app.js
  • (fabrika) → Express app oluşturur, route'ları monte eder

  • GET /api/health                 DB ping yanıtı (200 / 503)

ÖNEMLİ AKIŞLAR
================================================================

[POLLEN DATA AKIŞI]
  frontend fetchPollen() → GET /api/pollen
  → optionalAuth → enforceQuota('pollen') → pollen route handler
  → getPollenData() [tools/pollen.js]
    → fetchGooglePollen() [google.js] VEYA fetchOpenMeteoPollen() [openMeteo.js]
  → frontend: adaptLivePollen() → dashboard widget'ları

[CHATBOT AKIŞI]
  frontend sendChatMessage() → POST /api/chat
  → optionalAuth → enforceQuota('chat') → chat route handler
  → generateChatReply() [services/chat.js]
    → gatherContext() [RAG retriever + fetchPollenSummary PARALEL]
    → buildSystemPrompt() [prompts/chatPrompt.js]
    → Groq fallback döngüsü (GroqKeyManager)

[AUTH AKIŞI]
  register/login → JWT üret → localStorage.token
  Her istek → apiRequest() Bearer header ekle
  401 → auth:unauthorized event → AuthContext clearUser()

[KOTA AKIŞI]
  enforceQuota(resource) → resolveSubjectAndPlan()
    (giriş yapmış: user_id / misafir: IP adresi)
  → consume() [services/usage.js] atomik UPDATE
  → 429 QUOTA_EXCEEDED → quota:exceeded event → UsageContext → modal

NOTLAR
================================================================
• PAYMENT_MODE=simulate (varsayılan): Gerçek para hareketi YOK.
  Gerçek iyzico için: npm i iyzipay + .env PAYMENT_MODE=iyzico
• data/vector-store.json: RAG için gerekli (~2.5 MB vektör indeksi)
  Yoksa: node rag/build-index.js ile oluştur
• Groq modelleri: llama-3.3-70b-versatile (öncelik) → llama-4-scout → gpt-oss-120b → gpt-oss-20b
• Google Pollen API boykot süresi: 5 dk (auth/kota hatasında)
• Kota: anon=3/gün, free=10/gün, premium=sınırsız (pollen ve chat için)
